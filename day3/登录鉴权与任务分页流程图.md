# 登录鉴权与任务分页流程图

## 1) 登录与 Token 下发

```mermaid
flowchart TD
    A[前端: 输入用户名密码] --> B[POST /api/auth/login]
    B --> C[Gateway: 白名单放行 /api/auth/**]
    C --> D[auth-service: 校验账号密码]
    D --> E[签发 JWT]
    E --> F[返回 ApiResponse.data.token]
    F --> G[前端保存 TOKEN 到 localStorage]
```

## 2) 受保护接口访问（以任务搜索为例）

```mermaid
flowchart TD
    A[前端: loadTasks page,size] --> B[POST /api/tasks/search + Authorization Bearer Token]
    B --> C[Gateway JwtGlobalFilter]
    C --> D{Token 是否有效}
    D -- 否 --> E[返回 401]
    D -- 是 --> F[解析 uid username roles]
    F --> G[写入请求头 X-User-Id/X-Username/X-Roles]
    G --> H[转发到 task-service]
    H --> I[UserContextFilter 读取请求头]
    I --> J[写入 ThreadLocal UserContext]
    J --> K[TaskController.search]
    K --> L[TaskAppServiceImpl.search]
    L --> M[MyBatis-Plus Page page,size + selectPage]
    M --> N[返回 records,total,current,size]
    N --> O[前端渲染列表]
    O --> P[totalPage = ceil total/size]
    P --> Q[上一页/下一页触发下一次 loadTasks]
```

## 3) 关键说明

- Token 校验在 Gateway 做，微服务通常不再手动 `request.getToken()`。
- 微服务通过 `UserContext` 获取当前用户信息（来源于网关透传请求头）。
- 分页参数由前端传 `page/size`，后端返回 `records/total`，前端再计算总页数。
