## 项目描述（可直接放简历）

基于 Spring Cloud + Spring Boot + MyBatis-Plus + MySQL + Redis + OpenFeign + Gateway 构建任务协同平台，覆盖认证鉴权、任务管理、管理端看板与前端控制台，完成从网关统一鉴权到业务落地的全链路实现。  

- 设计并实现 Gateway 层 JWT 统一鉴权机制，对非白名单接口强制校验 Token；解析后透传用户信息头（`X-User-Id / X-Username / X-Roles`），并对管理端接口增加 `ADMIN` 角色拦截。  
- 在微服务侧实现 `UserContextFilter + ThreadLocal` 用户上下文方案，业务层通过 `UserContext` 直接获取当前用户，避免各服务重复解析 Token，统一处理 401/403 鉴权场景。  
- 实现任务模块核心能力：任务创建、详情查询、热榜查询与条件分页检索（关键词/状态/优先级区间），基于 MyBatis-Plus `Page(current,size)` 完成高效分页。  
- 在读链路落地 Redis ZSet 热榜缓存与定时重建策略，降低热点查询对数据库压力，提升高频场景下接口响应性能。
