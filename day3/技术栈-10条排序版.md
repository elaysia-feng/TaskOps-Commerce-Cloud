# 技术栈（按熟悉度排序，简历版）

1. **C / C++（熟悉）**：熟悉 C/C++ 语法与 STL，具备内存管理、常见并发/系统编程场景实践，能够使用 gdb 进行问题定位与调试。
2. **Java（熟悉）**：熟悉 Java 语法与面向对象编程，掌握集合（List/Map）、异常处理与常用 API，能够独立完成基础业务开发。
3. **Spring Boot（熟悉）**：掌握分层架构（Controller/Service/Mapper）、REST 接口设计、统一异常处理与拦截器链路实践。
4. **MySQL（掌握）**：具备表结构设计与索引基础，能够编写常见业务 SQL 并进行基础性能优化。
5. **MyBatis / MyBatis-Plus（掌握）**：熟悉 Mapper 开发模式，能够实现 CRUD、动态查询与分页；在项目中使用 MyBatis-Plus 的 `Page` 完成分页检索。
6. **Redis（掌握）**：掌握缓存设计、TTL 控制、PV 计数、固定窗口限流等常见场景，具备热点数据优化思路。
7. **认证与权限（掌握）**：熟悉 JWT + RBAC（用户-角色-权限）模型；在项目中实现 Gateway 统一鉴权、401/403 控制与用户上下文透传。
8. **Spring Cloud 微服务（了解并实践）**：具备 Gateway、OpenFeign、Nacos 的实际使用经验，完成认证服务、任务服务、管理服务拆分与调用。
9. **工程化与环境（掌握）**：熟悉 Linux / Git / Maven / Docker Compose，具备项目构建、版本管理、联调部署与基础问题排查能力。
10. **定时任务与实时能力（了解并实践）**：使用 Spring Scheduling 处理周期任务；了解 WebSocket 实时推送机制与看板类场景实现方式。
