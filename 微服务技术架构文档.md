# TaskOps-Cloud 微服务技术架构文档

## 一、项目概述

- **项目名称**: taskops-cloud
- **版本**: 1.0.0-SNAPSHOT
- **Java版本**: 17
- **架构模式**: Spring Cloud 微服务架构

---

## 三、微服务清单

| 服务 | Artifact ID | 功能描述 |
|------|-------------|----------|
| API网关 | gateway-service | 请求路由、鉴权、流量控制 |
| 认证服务 | auth-service | 用户认证、授权、JWT令牌 |
| 任务服务 | task-service | 任务创建、查询、管理 |
| 订单服务 | order-service | 订单创建、状态管理 |
| 支付服务 | pay-service | 支付、退款流水 |
| 钱包服务 | wallet-service | 账户余额、充值 |
| 通知服务 | notification-service | 消息推送、邮件通知 |
| 管理服务 | admin-service | 后台管理功能 |
| AI代理服务 | ai-proxy-service | AI对话代理、流式响应 |
| 公共模块 | cloud-common | 公共实体、工具类 |

## 五、技术亮点汇总

### 5.1 基础设施
| 技术 | 用途 |
|------|------|
| **Alibaba Nacos** | 服务注册中心 + 配置中心 |
| **Alibaba Sentinel** | 流量控制、熔断降级 |
| **Spring Cloud LoadBalancer** | 客户端负载均衡 |

### 5.2 服务通信
| 技术 | 用途 |
|------|------|
| **OpenFeign** | 声明式服务调用 |
| **RabbitMQ (AMQP)** | 异步消息队列 |

### 5.3 分布式能力
| 技术 | 用途 |
|------|------|
| **Seata** | 分布式事务 (order-service, pay-service) |
| **JWT** | 身份令牌 (gateway, auth) |

### 5.4 数据存储
| 技术 | 用途 |
|------|------|
| **MyBatis Plus** | ORM框架 |
| **MySQL** | 关系型数据库 |
| **Redis** | 缓存 + Session共享 |

### 5.5 AI能力
| 技术 | 用途 |
|------|------|
| **Spring AI OpenAI** | AI对话能力 |
| **Spring WebFlux** | 响应式流式响应 |

### 5.6 文档与工具
| 技术 | 用途 |
|------|------|
| **springdoc OpenAPI** | Swagger API文档 |
| **Lombok** | 简化代码 |
